About This Project

  • Technology: Ray marching with signed distance functions (SDFs)
  • Math: CORDIC algorithm (shifts and adds only - no multiplications!)
  • Output: VGA 640x480 @ 60Hz with 256 colors (3-3-2 RGB)
  • Animation: Rotating camera using Minsky circle algorithm
  • Platform: TinyTapeout ASIC (2x2 tiles, ~7000 gates)

Technical Details

  • Resolution: 640x480 pixels
  • Color Depth: 8-bit (256 colors)
  • Fixed-Point: Q8.8 format
  • Ray Steps: Maximum 16 iterations
  • CORDIC Iterations: 3-4 (creates faceted aesthetic)
  • Rendering Time: ~46ns per pixel @ 65MHz
